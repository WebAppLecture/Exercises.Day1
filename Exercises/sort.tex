\RequiredExercise{Zahlen sortieren}
%
\par Über eine \jkey{while}-Schleife soll solange ein \jfunc{prompt} angezeigt werden, wie der Benutzer eine gültige Zahl eingibt (d.h. Abbruch der Schleife bei ungültiger Eingabe). Die Zahlen sollen in einem Array \jvar{zahlen} gespeichert werden. Nach dem Abbruch der \jkey{while}-Schleife soll das eingegebene Array zahlen aufsteigend sortiert werden.
%
\par Abschließend soll das Array über \jfunc{alert} ausgegeben werden. Benutzen Sie hierzu \jfunc{zahlen.toString}.
